# PWA Icons

Generate icons from the Be[X] logo:

- icon-192.png (192x192)
- icon-512.png (512x512)
- icon-maskable-512.png (512x512, with padding for safe zone)

Source SVG: `icon.svg` — use it to generate the PNGs (e.g. via Inkscape, ImageMagick, or an online tool).

For maskable icons, ensure the logo is within the inner 80% safe zone (leave 10% padding on each side).
